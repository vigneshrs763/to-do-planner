# 📝 To-Do Planner

A simple and user-friendly task management application that helps users organize, track, and manage daily tasks efficiently.

## 🚀 Features

- Add new tasks
- Edit existing tasks
- Delete tasks
- Mark tasks as completed
- Simple and clean user interface
- Responsive design

## 🛠️ Tech Stack

- HTML
- CSS
- JavaScript


## 🎯 Future Improvements

- User Authentication
- Task Categories
- Due Dates and Reminders
- Dark Mode
- Database Integration
